Slang Keywords

Here is your reference guide to scripting in bhidu-lang. Trade standard JS/TS operators for local slang.

Bhidu Slang JS Equivalent What it does
chalu kar bhidu *(implicit)* Program opening syntax. Essential at the top of the file!
khatam bhidu *(implicit)* Program closing syntax. Must close your file with this.
bhidu ye hai let / var Declare a variable (e.g. bhidu ye hai counter = 0;)
bhidu bolta hai(...) console.log(...) Outputs expressions, variables or calculations to the console.
sahi bhidu true Boolean true state.
galat bhidu false Boolean false state.
khali bhidu null Empty reference or null variable state.
agar bhidu(...) if(...) Execute block if condition evaluates to true.
warna agar bhidu(...) else if(...) Execute fallback check block.
warna bhidu else Default conditional execution block.
jab tak bhidu(...) while(...) Iterative block that runs as long as the condition holds.
bas kar bhidu break Exits loops immediately.
agli baar bhidu continue Skips the current loop step and starts the next iteration.

Syntax Guide

Quick structures to help you get writing in no time.

⚡

Main File Structure

Every file must end with .bhidu and encapsulate statements inside chalu kar bhidu and khatam bhidu.

🔒

Variables & Reassignment

Initialize variables using bhidu ye hai x = 5;. Reassignment doesn't require prefix keywords: x = 10;.

🛠️

Blocks & Scoping

Standard bracket scopes { ... } isolate variables. Nested blocks support variable shadowing out of the box.

CLI Commands Reference

How to run, parse, and execute programs from your host terminal once installed.

bhidu run <file>

Compiles, parses, and executes the target script directly.

node dist/cli.js run test.bhidu

bhidu repl

Launches interactive shell. Preserve environment states and write line-by-line statements.

node dist/cli.js repl

bhidu ast <file>

Prints out the parsed Abstract Syntax Tree (AST) representation in raw JSON.

node dist/cli.js ast test.bhidu
